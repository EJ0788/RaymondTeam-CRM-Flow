Funnel Flow Map

Global Automation Rules
Timeline tags are mutually exclusive
Applying a new timeline tag auto-removes all others. Old Smartplan stops. New one starts.
Primary tags are mutually exclusive
RT|Trust overrides RT|Buyer or RT|Seller. One primary tag per lead at all times.
Hand Raised = universal circuit breaker
Any intent signal (email reply, saved listing, form fill, booking click, OTT pixel return) applies RT|Hand Raised, pauses ALL Smartplans, moves lead to Hand Raised stage, fires agent notification.
Re-entry prevention
If lead is already in an active Smartplan, a second opt-in only logs the new source tag. Does not re-enroll or restart sequences.
Complexity tags are additive
RT|Family Conflict, Rehab Risk, Alignment Risk, Low Equity can stack. They fire internal agent alerts only — never lead-facing sequences.
Research drip = end of line
Leads completing a sequence with zero engagement receive RT|Research. Low-frequency evergreen content only. No re-enrollment until intent signal fires.
Authority tags are pipeline-scoped
RT|Authority – Confirmed/Pending only valid inside Trust/Probate pipeline. Applying elsewhere triggers admin review flag.
